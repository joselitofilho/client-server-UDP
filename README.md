# client-server-UDP

This project implements the communication between client and server via User Datagram Protocol (UDP).

**Table of Contents**

- [Build](#build)
- [Run](#run)

## Build

```bash
$ make
```

Server:
```bash
$ make server
```

Client:
```bash
$ make client
```

## Run

Server:
```bash
$ bin/server
```

Client:
```bash
$ bin/client
```
